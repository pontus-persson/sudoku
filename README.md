# Sudoku

Made this simple Sudoku solver since I saw some youtube video about hard sudoku puzzles. Note that I have never solved a sudoku, was just interested in how the constraints could be implemented in code.


## Tech Stack

**Client:** Typescript

**Build:** Node, Webpack


## Installation

Install npm

```bash
  npm install
```

Run development

```bash
  npm run dev
```

Run for prod

```bash
  npm run deploy
```
